## Prerequisites

### Backend
- Python 3
- Django
- Daphne (ASGI server)
- Channels (for WebSocket support)

### Frontend
- Node.js & npm or yarn
- React
- Vite

## Getting Started

### Backend
```bash
daphne -p 8000 name_of_your_app.asgi:application
```

### Frontend 
```bash
yarn create vite your-app-name --template react
```
```bash
cd your-app-name
```
```bash
yarn install

```
```bash
yarn dev
```
